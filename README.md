# boolean_circuit

Simply run the bash script file as
 ```bash
./run.sh
```
Doing so will run two python scripts 'gen_truth_table.py' and 'gen_circuit.py'.
These two scripts will fetch the following inputs from the 'inputs' directory
 - EXTRA2.bool
 - EXTRA1.bool
 - test1.bool
 - test2.bool
 - test3.bool
 - test4.bool
 - test5.bool
 - compute5.cir
 - compute1.cir
 - compute2.cir
 - compute3.cir
 - compute4.cir
 
 and write the following output files in the 'outputs' directory
 - EXTRA2.bool.out
 - EXTRA1.bool.out
 - test1.bool.out
 - test2.bool.out
 - test3.bool.out
 - test4.bool.out
 - test5.bool.out
 - compute5.cir.out
 - compute1.cir.out
 - compute2.cir.out
 - compute3.cir.out
 - compute4.cir.out
